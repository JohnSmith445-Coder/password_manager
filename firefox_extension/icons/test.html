Kasm Secure Browser Extension Test

How to Test the Extension

  1. Make sure the Kasm Workspace is running
  2. Ensure the Firefox extension is installed
  3. Right-click on any of the links below
  4. Select "Open in Secure Browser" from the context menu
  5. The link will open in a Chrome container within Kasm Workspace
  6. A new tab will automatically open with the secure container interface
  7. Once the container is fully loaded, it will navigate to your selected URL

Test Links

Google Wikipedia GitHub Reddit YouTube

Note

If the extension doesn't work:

  • Check that Kasm Workspace is running on localhost:5000
  • Verify that you're logged into Kasm Workspace
  • Make sure the Chrome container is properly configured in Kasm Workspace
  • Check the browser console for any error messages
